## Install instructions

If ou wish to install a provided version of all needed manifests, you can find them here:
[https://raw.githubusercontent.com/FabioTavernini/spendemon/refs/heads/main/deploy/spendemon.yaml](https://raw.githubusercontent.com/FabioTavernini/spendemon/refs/heads/main/deploy/spendemon.yaml)

So the apply command becomes:
```sh
kubectl apply -f https://raw.githubusercontent.com/FabioTavernini/spendemon/refs/heads/main/deploy/spendemon.yaml
```
