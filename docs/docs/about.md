## About Spendemon

Spendemon helps teams get a fast, readable view of Kubernetes usage and estimated workload cost without needing a full billing platform.

## Workflow

The project focuses on a practical workflow:

- connect one or more Prometheus endpoints
- discover clusters, namespaces, and pods
- estimate cost from requested CPU, memory, and storage
- make settings editable through the UI and `settings.yaml`

## Stack

The application is built with Next.js, TypeScript, Tailwind CSS, and Prometheus as the data source.
